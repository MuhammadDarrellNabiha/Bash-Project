# Number Counter

My second Bash scripting project.

This project was created to practice Bash scripting fundamentals by building a simple interactive number counter with a command-line menu.

---

## Features

- Count from 0 to 100
- Count from 100 to 1
- Display even numbers
- Display odd numbers
- Interactive menu
- Exit option

---

## Skills Practiced

- Variables
- While Loop
- Until Loop
- Case Statement
- Arithmetic Expansion
- User Input (`read`)
- Conditional Statements
- Interactive CLI Program
