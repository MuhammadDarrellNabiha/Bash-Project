# Bash Project

A collection of Bash scripting projects created while learning Bash from the basics to real-world automation.

The goal of this repository is to document my learning journey, improve problem-solving skills, and build a strong foundation before moving on to Python automation.

---

## Projects

### Project 1 — File Organizer

Organize files into folders based on their file extensions.

**Topics**

- Variables
- If Statement
- For Loop
- File Operations
- mkdir
- mv

---

### Project 2 — Number Counter

Interactive CLI program for practicing loops and control flow.

**Topics**

- While Loop
- Until Loop
- Case Statement
- Arithmetic Expansion
- User Input

---

### Project 3 — Directory Scanner

A menu-driven Bash application that scans directories and displays useful information.

**Features**

- Show Files
- Show Directories
- Count Files
- Count Directories
- Search Files

**Topics**

- Functions
- Positional Parameters
- Case Statement
- While Loop
- File Test Operators
- Parameter Expansion
